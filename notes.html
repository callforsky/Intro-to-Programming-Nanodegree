HTML BASICS

Formatting

Similar to Markdown, HTML uses tags to change the format of text. Multiple spaces are recognized as exactly one space, and a simple Return will not start a new line. There are a couple of ways to start a new line in HTML. The first way is to add < br > at the end of the line. The second way is to wrap the paragraph with < p > and < /p >. The thrid way is to wrap the block of texts with < div > and < /div >. There is a big difference between < div > and < p >. Simply said, the former is a block tag, while the latter a inline tag. For now, we should not worry too much about their difference, as we should learn it later in the course. Below are the formatting tags we have covered so far in this course:

(1) Bold

To make text bold in HTML, wrap the text with < b > and < /b > tags

(2) Italics

To make text italic , wrap the text with < em > and < /em > tags

Functioning

HTML tags can also do some functional jobs other than formatting texts. You can insert a link, image, or whatever media, into a HTML document through tags. Below are the functional tags that have been covered in this course so far:

(1) Insert a link

To make a text hyperlink, like this link to Udacity, wrap the text with <a href="url"> and </a>, where url is the http address like www.google.com.

(2) Insert an image

To insert an image, simply insert <img src="url"> tags. It is recommend to add some error handling attributes in case of failing to load the image from the source. To do so, add alt attributes to the image tag: <img src="url" alt="text">, where text is the text that you want to display in place of the image if the image does not show up as expected.
